# Hospital-Management-System
Basically it's a Database Management System Project, Done using the MongoDB database.
Technologies Used:
HTML, CSS, JAVASCRIPT, QUERY and JSON.
Database used is MONGODB.

**MONGODB** :- MongoDB works as a NoSQL and document-oriented database that stores data in a flexible, JSON-like format.
Instead of rows and columns (like SQL), it uses collections and documents:

Database: A container for collections (similar to schema in SQL).

Collection: A group of documents (similar to a table).

Document: A record stored in JSON-like key-value pairs.

**More About Project**:
-----------------------------------------------------------------------------------------------------------------------------------------------------------------
On the Home Page user has to select it's Category Either Patient or Admin. 

Patient Page Dashboard includes:- Book Appointment, My Appointments, Account Detail and Available Slot

Hospital Management Dashbord includes :-Overview, Patient, Doctors, Appointments
